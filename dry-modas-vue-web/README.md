## 📥 download

[Node.Js](https://nodejs.org/en/): `v16.19.0` 

### 💻 setup

```php
# instale as dependências
$ npm install -f
```

## ▶ starting

```php
# executando o comando'
$ ./run

# --- ou ---

$ npm run dev

# aplicação iniciará em <http://127.0.0.1:5173/#/>
```
